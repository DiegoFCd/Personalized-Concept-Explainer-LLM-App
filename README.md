## 📌 Project Overview

This project leverages Large Language Models (LLMs) to generate concise and personalized explanations of technical concepts.
Using Prompt Engineering, LangChain, Gemini 2.5 Flash and Gradio, the application adapts explanations to the user's background and learning objectives.
<br>

## 🎯 Business Objective

Generic explanations often ignore the learner's prior knowledge.
The objective of this project is to provide contextualized explanations that make complex concepts easier to understand.
<br>

## 🛠 Technologies Used
- Python
- LangChain
- Gemini 2.5 Flash
- PromptTemplate
- Gradio
- dotenv
<br>

## ⚙ Workflow
```text
User Input
↓
Prompt Template
↓
Gemini 2.5 Flash
↓
Personalized Explanation
```
